Fonctionnalités

  • Recherche par ISBN — validation checksum, conversion ISBN-13/10
  • Sources bibliographiques multi-moteurs : BnF (recommandée), OpenLibrary, Google Books
  • Enrichissement automatique avec fallback (champs critiques : titre, auteur, éditeur, pages)
  • Gestion de couverture via OpenLibrary Covers — badge de source
  • Suggestions de thème et sous-thème via IA (Claude Haiku)
  • Génération de fiche de lecture via IA (Claude Sonnet)
  • Intégration Notion : pré-vérification ISBN, création et mise à jour de page
  • Synchronisation automatique du schéma Notion (propriétés manquantes créées)
  • Badges de source par champ (ISBN, Notion, IA) + journal de recherche en popover
  • Détection automatique des éditions spéciales et collections
  • Support iOS Shortcuts via paramètre URL ?isbn=XXXXX
  • Interface dark mode, responsive mobile

Guide d'initialisation

  1. Base Notion
    Créer une page de base de données vierge dans Notion (sans modèle). Copier l'identifiant de 32 caractères depuis l'URL de la page. L'application crée automatiquement toutes les propriétés nécessaires lors du premier enregistrement.
  2. Intégration Notion
    Aller sur notion.so/my-integrations → "New integration". Nommer l'intégration (ex. "GEBIBLIPE"), sélectionner l'espace de travail, copier le token (ntn_… ou secret_…). Dans Notion, ouvrir la base → "…" → "Ajouter des connexions" → sélectionner l'intégration.
  3. Proxy Cloudflare (recommandé)
    Les navigateurs bloquent les appels directs à l'API Notion (CORS). Créer un compte Cloudflare gratuit → Workers → nouveau worker avec un script de proxy CORS vers api.notion.com. Copier l'URL du worker (ex. https://xxx.workers.dev).
  4. Clé API Anthropic (optionnelle)
    Pour les fonctionnalités IA (suggestion de thème, génération de fiche). Créer un compte sur console.anthropic.com → API Keys. Le token commence par sk-ant-.
  5. Configuration dans l'appli
    Cliquer sur ⚙ configuration en bas de page. Renseigner les champs (proxy, token, database ID, clé Anthropic). Cliquer sur "Vérifier et enregistrer" — l'application synchronise automatiquement le schéma de la base Notion.
